# BinaryTextAdventure 

BinaryTextAdventure ist ein kleine schulisches Projekt zur Implementierung der Klasse BinaryTree des Landes NRW. 

## Was ist BinaryTextAdventure genau? 

In BinaryTextAdventure durchlebt man eine kurze Geschichte in Form eines TextAdventures. Ein Editor ermöglicht jedoch auch, eigene Geschichten zu schreiben.  

Fachdidaktisch wird die Geschichte als binärer Baum gespeichert. Jeder Knoten kann entweder "Ereignis" oder "Ende" sein; und zu jedem Ereignis gibt es zwei Wahlmöglichkeiten. Zudem umfasst das Projekt viele Konzepte, die im Informatikunterricht der Q1 und Q2 etabliert werden. 

## Wie wird BinaryTextAdventure eingesetzt? 

Einfach die TextAdventure Klasse starten. 

## Wie kann ich BinaryTextAdventure schulisch einbinden? 
Didaktische Schwerpunkte sind/können sein:
- Exception Handling
- ENUM
- Interfaces
- Abstrakte Klassen
- Datei Input und Output mit eigenem Format
- Sound Ausgabe
